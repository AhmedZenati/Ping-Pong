# 🏓 Ping Pong

A simple browser-based ping pong game built with HTML, CSS, and JavaScript — no dependencies, single file.

## Features

- **vs Computer** with 3 difficulty levels (Easy / Medium / Hard)
- **2 Players** on the same device
- Mobile friendly — touch controls + drag support
- First to **7 points** wins

## Controls

| | Player 1 | Player 2 (2P mode) |
|---|---|---|
| Keyboard | `W` / `S` or `↑` / `↓` | `↑` / `↓` |
| Mobile | Drag on left half of screen | Drag on right half |
| Buttons | ▲ ▼ on-screen | ▲ ▼ on-screen |

Press **Space** or tap the canvas to serve.

## Deploy to Vercel (free)

1. Rename `pingpong.html` → `index.html`
2. Put it in a folder
3. Go to [vercel.com](https://vercel.com) and sign up (free)
4. Drag & drop the folder into the Vercel dashboard
5. Get a live `.vercel.app` URL instantly

Or use the CLI:

```bash
npx vercel
```

## Tech Stack

- HTML5 Canvas
- Vanilla CSS & JavaScript
- No frameworks, no dependencies
